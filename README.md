## Introduction
This website serves as a platform for me to share my personal projects, connect with others through my social media, and provide a glimpse into who I am. It's built using basic HTML, CSS, and JavaScript.

## Features
1. **Social Links**: You can find links to my various social media profiles.
2. **Depth First Search Maze Solver**: A fun maze-solving algorithm implemented in JavaScript.
3. **Personal Bio**: Get to know a little more about me, my interests, and my background.
4. **p5 Animation**: Experience a cool animation created using the p5.js library.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- p5.js (for the animation)
  
## Setup and Usage
1. Clone the repository to your local machine.
   ```bash
   git clone https://github.com/nathdilla/nathdilla.github.io.git
   ```
2. Navigate to the project directory.
   ```bash
   cd nathdilla.github.io
   ```
3. Open `index.html` in your preferred web browser to view the website locally.
